Impact of Lattitude VS. Weather Patterns

This projects goal was to analyze the change in weather as you get closer to the equator. To arrive at the analysis, data was pulled from the OpenWeatherMap API to assemnle a data set on several hundred cities.

Once the dataset was put together, Matplotlib was used to plot different aspects of the weather versus the lattitude. A few of the different things that we looked at were: humidity, cloudiness, wind speed and temperature. This website serves to show the source data as well as all visualizations as a result of the data. You will also see detailed descriptions of each visual corresponding to the trends seen in the study.
